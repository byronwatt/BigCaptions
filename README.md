# BigCaptions

BigCaptions is an iOS app designed for deaf and low-vision users, providing real-time, high-visibility live captioning.

## Features
- **Voice-to-Text:** Real-time transcription using Apple's Speech Recognition.
- **Enhanced Visibility:** Large, bold fonts with high contrast (white text on black).
- **Customizable Font Size:** Adjust the font size from 20 to 100 in the settings or just pinch to zoom, just a handful of fonts are available (I'm not a UI expert!),... "Inter" is the best.
- **pauses start sentences/paragraphs:** a small pause starts a new line, a medium pause adds a half-height blank line, a long pause adds a faint bar with the tiem of day and a full blank line (short/med/long is configurable)
- **local/remote transcription** i've only tested local transcription - have not tested using remote transcription.
- **show/hide top bar** 
- **configurable list of words** you can encourage the speech to text to select particular words if they are close.  e.g. adding peoples names might help.  oh,... i think there's a double tap to add the word to the list.  i forgot about that !  it hasn't been tested much.

## Permissions
The app requires the following permissions to function:
- **Microphone:** To capture audio for transcription.
- **Speech Recognition:** To process the audio into text.

## Power usage
The app uses a fair amount of cpu,... apparently switching the speech to text engine to 'remote' might save some battery.
there's a 'session info' screen that tries to estimate power usage and time remaining.  while the app is active it tries to measure battery drop and use historical data to estimate time left.
